Interface class for writing coefficients as a function of N independent variables to a file. This class is used instead of a single templated free function to allow multi-\/arrays of different sizes to be created using the same interface. NOTE\+: The possibility of using a single templated implementation for arbitrary multi-\/array size should be investigated in the future.

\doxysubsection{Detailed Description}
Function to compute first-\/order approximation of proper-\/time rate for one-\/way Doppler observable. 

Function to compute first-\/order approximation of proper-\/time rate for one-\/way Doppler observable. Assumes a single static mass generating the gravity field (Schwarzschild metric) and a c$^\wedge$-\/2 expansion of proper time rate function.

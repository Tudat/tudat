\doxysubsection{Detailed Description}
Class to compute a phenomenological dissipative torque model. 

Class to compute a phenomenological dissipative torque model, with the torque proportional to the angular velocity deviation w.\+r.\+t. the mean rotation (about body-\/fixed z-\/axis).

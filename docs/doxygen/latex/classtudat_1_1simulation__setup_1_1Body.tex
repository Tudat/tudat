\doxysubsection{Detailed Description}
\mbox{\hyperlink{classtudat_1_1simulation__setup_1_1Body}{Body}} class representing the properties of a celestial body (natural or artificial). 

\mbox{\hyperlink{classtudat_1_1simulation__setup_1_1Body}{Body}} class representing the properties of a celestial body (natural or artificial). By storing all properties of bodies (ephemeris, rotation, gravity, etc.) in a set of body objects, the simulation environment can be defined in a clear and modular way. To create body objects, the \mbox{\hyperlink{createBodies_8h_source}{create\+Bodies.\+h}} function provides a range of functionality. The \mbox{\hyperlink{createAccelerationModels_8h_source}{create\+Acceleration\+Models.\+h}} file provides functions to use body objects to create acceleration objects.

\doxysubsection{Detailed Description}
Solar activity parser class. 

This class implements a fixed width parser specifically designed for parsing solar activity files (space weather data) in the format of \href{http://celestrak.com/SpaceData/sw19571001.txt}{\texttt{ http\+://celestrak.\+com/\+Space\+Data/sw19571001.\+txt}}.

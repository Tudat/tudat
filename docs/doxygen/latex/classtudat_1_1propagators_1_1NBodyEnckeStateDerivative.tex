Class for computing the state derivative of translational motion of N bodies, using an Encke propagator. 

Class for computing the state derivative of translational motion of N bodies, using an Encke propagator. The Encke propagator propagates the Cartesian deviation from an ideal (pre-\/defined) Keplerian orbit. See e.\+g. Wakker, astro II for mathematical details.

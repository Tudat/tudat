\doxysubsection{Detailed Description}
Wrapper class that contains the relevant hardware systems of a vehicle. 

Wrapper class that contains the relevant hardware systems of a vehicle. Not all member objects need to be set; nullptr member objects denote that the vehicle does not contain the associated hardware.

\doxysubsection{Detailed Description}
Interface class for the estimation of a constant absolute or relative observation bias. 

Interface class for the estimation of a constant absolute or relative observation bias (at given link ends and observable type). Unlike most other \mbox{\hyperlink{classtudat_1_1estimatable__parameters_1_1EstimatableParameter}{Estimatable\+Parameter}} derived classes, this class does not have direct access to the class (Constant\+Observation\+Bias/\+Constant\+Relative\+Observation\+Bias) used in the simulations for the observation bias. This is due to the fact that the Constant\+Observation\+Bias and Constant\+Relative\+Observation\+Biases class are templated by the observable size, while this class is not.

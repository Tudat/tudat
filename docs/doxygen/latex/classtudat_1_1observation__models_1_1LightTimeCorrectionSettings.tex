\doxysubsection{Detailed Description}
Base class for light-\/time correction settings. 

Base class for light-\/time correction settings. This class is not used for calculations of corrections, but is used for the purpose of defining the light time correction properties. The create\+Light\+Time\+Corrections function produces the classes that calculate the actual corrections, based on settings in instances of derived \mbox{\hyperlink{classtudat_1_1observation__models_1_1LightTimeCorrectionSettings}{Light\+Time\+Correction\+Settings}} classes.

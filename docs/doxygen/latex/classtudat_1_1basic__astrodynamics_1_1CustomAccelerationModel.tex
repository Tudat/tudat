Update member variables used by the acceleration model. 

Updates member variables used by the acceleration model. In the case of acceleration models containing varying parameters, function-\/pointers returning such a parameter (for instance the Cartesian state of a body) will be set as a member variable. This function evaluates such function-\/pointers and updates member variables to the \textquotesingle{}current\textquotesingle{} values of these parameters. Only these current values, not the function-\/pointers are then used by the \mbox{\hyperlink{classtudat_1_1basic__astrodynamics_1_1CustomAccelerationModel_ab3dcc99da7bb98850550205f8395e023}{get\+Acceleration()}} function.

N.\+B.\+: This pure virtual function must be overridden by derived classes!

Class to manage and execute the numerical integration of variational equations of a dynamical system in a combination of single and multiple arcs

Class to manage and execute the numerical integration of variational equations of a dynamical system, in addition to the dynamics itself, in a combination of single and multiple arcs. In this class, the governing equations are set once, but can be re-\/integrated for different initial conditions using the same instance of the class.

\doxysubsection{Detailed Description}
Class for variations in spherical harmonic coefficients, directly interpolated from tables. 

Class for variations in spherical harmonic coefficients, directly interpolated from tables. Maps of variatons of cosine and sine coefficients as a function of time are provided, as well as the type of interpolator that is to be used, from which interpolated variations are then returned at any time. The class may also be used for defining the complete gravity field (i.\+e. not only the variations) as a function of time when the constant coefficient contribution (i.\+e. not from a \mbox{\hyperlink{classtudat_1_1gravitation_1_1GravityFieldVariations}{Gravity\+Field\+Variations}}) is zero for all degrees and orders.

\doxysubsection{Detailed Description}
Class to calculate first order relativistic light time correction (Shapiro time delay) due to a set of point masses. 

Class to calculate first order relativistic light time correction (Shapiro time delay) due to a set of point masses. This class has the properties (mass and position) of the gravitating bodies as member variables. It receives the states and times of receiver and transmitter as input each time a light time is to be calculated. NOTE\+: If the perturbing body is equal to the transmitting or receiving body, the transmission or reception time, respectively, are used to evaluate this perturbing body\textquotesingle{}s state when computing the correction. For all other bodies, the mid-\/way point of the light time is used.

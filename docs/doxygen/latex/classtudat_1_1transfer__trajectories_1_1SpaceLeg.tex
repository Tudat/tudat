\doxysubsection{Detailed Description}
Space Leg base class. 

A base class for calculating the required impulses for a high-\/thrust, patched-\/conics space leg. It inherits from the mission leg base class. The space leg class is inherited by different trajectory models, for departure legs and swing-\/by legs. One additional variable is declared here.

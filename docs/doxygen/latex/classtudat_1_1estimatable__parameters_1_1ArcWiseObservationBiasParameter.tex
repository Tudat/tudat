\doxysubsection{Detailed Description}
Interface class for the estimation of an arc-\/wise constant absolute or relative observation bias. 

Interface class for the estimation of aan arc-\/wise constant absolute or relative observation bias (at given link ends and observable type). Unlike most other \mbox{\hyperlink{classtudat_1_1estimatable__parameters_1_1EstimatableParameter}{Estimatable\+Parameter}} derived classes, this class does not have direct access to the class (Constant\+Arc\+Wise\+Observation\+Bias/ Constant\+Relative\+Arc\+Wise\+Observation\+Bias) used in the simulations for the observation bias. This is due to the fact that the Constant\+Arc\+Wise\+Observation\+Bias and Constant\+Relative\+Arc\+Wise\+Observation\+Bias class are templated by the observable size, while this class is not.

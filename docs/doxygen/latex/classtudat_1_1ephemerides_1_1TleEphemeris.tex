\doxysubsection{Detailed Description}
\mbox{\hyperlink{classtudat_1_1ephemerides_1_1Ephemeris}{Ephemeris}} derived class that calculates the Cartesian position as a function of time assuming a two-\/line elements based orbit.

\mbox{\hyperlink{classtudat_1_1ephemerides_1_1Ephemeris}{Ephemeris}} derived class that calculates the Cartesian position as a function of time assuming a two-\/line elements based orbit. The class supports state propagation for Earth-\/orbiting satellites, given the set of two-\/line elements (TLE) that form the parameters for the SGP and SDP models. Currently, the class supports target reference frames with their origin at the center of the Earth, and either J2000 or ECLIPJ2000 orientation.

\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Independent\+Variable\+Type, typename Dependent\+Variable\+Type$>$\newline
class tudat\+::interpolators\+::\+Jump\+Data\+Linear\+Interpolator$<$ Independent\+Variable\+Type, Dependent\+Variable\+Type $>$}

Linear interpolation class with discrete jumps in data. 

This class is used to perform linear interpolation in a single dimension from a set of data in independent and dependent variables. The dependent variables contain discrete jumps at specified locations. The jumps are identified by a maximum allowed deviation between two dependent variable data points. The jump in data is fixed to a user-\/supplied value when a jump is identified.

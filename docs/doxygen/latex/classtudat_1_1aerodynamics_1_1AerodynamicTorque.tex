Get aerodynamic torque. 

Returns the aerodynamic torque. All data required for the computation is taken from member variables, which are set to their latest values by the last call of the update\+Members function. The returned torque is in the same reference frame as the aerodynamic coefficients, with the coefficients assumed to be in positive direction in the frame. \begin{DoxyReturn}{Returns}
Aerodynamic torque.

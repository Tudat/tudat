Detailed Description

Class to proivide settings for typical relativistic corrections to the dynamics of an orbiter.

Class to proivide settings for typical relativistic corrections to the dynamics of an orbiter: the Schwarzschild, Lense-Thirring and de Sitter terms. An excellent introduction to these models is given in 'General relativity and Space Geodesy' by L. Combrinck (2012).

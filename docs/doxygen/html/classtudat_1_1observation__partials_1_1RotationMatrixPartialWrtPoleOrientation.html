Detailed Description

Class to calculate a rotation matrix from a body-fixed to inertial frame w.r.t. a constant pole right ascension and declination

Class to calculate a rotation matrix from a body-fixed to inertial frame, as computed by the SimpleRotationalEphemeris class, w.r.t. the constant pole right ascension and declination

22  * representation of the (out-of-plane) elevation angle, phi (in spherical coordinates), of a
23  * thrusting spacecraft. This oscillating function can be used to approximate a
24  * continuous-thrust trajectory (also referred to as a low-thrust trajectory), when combined
25  * with a function which represents the radial position, r (in spherical coordinates), of a
26  * thrusting spacecraft. The spherical coordinate system that is used for the calculations and
27  * the descriptions is taken from Novak and Vasile [2011].

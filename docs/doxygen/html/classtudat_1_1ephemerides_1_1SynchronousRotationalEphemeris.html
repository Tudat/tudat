Detailed Description

Class to define a fully synchronous rotation model for a body.

Class to define a fully synchronous rotation model for a body. The body-fixed x-axis always points directly to the central body, and itts z-axis is along the orbital angular momentum vector (r x v). NOTE: The time derivative of the rotation matrix is not yet implemented.

Detailed Description

Class to compute typical relativistic corrections to the dynamics of an orbiter.

Class to compute typical relativistic corrections to the dynamics of an orbiter. The model allows the inclusion of the Schwarzschild, Lense-Thirring and de Sitter terms (any one of them may be turned on and off). An excellent introduction to these models is given in 'General relativity and Space Geodesy' by L. Combrinck (2012).

Update member variables used by the relativistic correction acceleration model.

Updates member variables used by the relativistic correction acceleration model. Function pointers to retrieve the current values of quantities from which the acceleration is to be calculated are set by constructor. This function calls them to update the associated variables to their current state.

Detailed Description

template<typename IndependentVariableType, typename DependentVariableType>
class tudat::numerical_quadrature::GaussianQuadrature< IndependentVariableType, DependentVariableType >

Gaussian numerical quadrature wrapper class.

Numerical method that uses the Gaussian nodes and weight factors to compute definite integrals of a function. The Gaussian nodes and weight factors are not calculated, but read from text files. The number of nodes (or weight factors) has to be at least n = 2. The current text files contain tabulated values up to n = 64.

Detailed Description

Class for propagation stopping conditions settings: stopping the propagation after a fixed amount of CPU time.

Class for propagation stopping conditions settings: stopping the propagation after a fixed amount of CPU time. Note that the propagator will finish a given time step, slightly surpassing the defined final CPU time.

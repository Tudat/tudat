Detailed Description

ConicalFrustum class.

Class that defines the conical frustum shape. The independent variables are the fraction of the total length of the frustum and the circumferential angle, respectively. Parameters are the initial radius, the length and the the cone half angle, respectively.

Get surface derivative on conical frustum.

Retrieves the derivatives of the surface point with respect to the two independent variables. For powerOfLengthFractionDerivative = 1 and powerOfAzimuthDerivative = 2 the function returns:

\[ \frac{ d^{ 3 } ( x, y, z ) } { du * dv^{ 2 } } \]

( with u and v the length fraction and azimuth angle ).

Parameters
lengthFractionLength fraction.
azimuthAngleAzimuth angle.
powerOfLengthFractionDerivativePower of derivative with respect to length fraction.
powerOfAzimuthAngleDerivativePower of derivative with respect to azimuth angle.
Returns
Surface derivative on conical frustum.

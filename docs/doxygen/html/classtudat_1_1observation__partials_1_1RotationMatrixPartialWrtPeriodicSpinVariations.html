Detailed Description

Class to calculate a rotation matrix from a body-fixed to inertial frame w.r.t. the periodic spin variations.

Class to calculate a rotation matrix from a body-fixed to inertial frame, as computed by the PlanetaryRotationModel class, w.r.t. the periodic spin variations.

Detailed Description

Interface class for estimation of a body's time-dependent (arcwise constant) empirical accelerations.

Interface class for estimation of a body's time-dependent (arcwise constant) empirical accelerations. Interfaces the estimation with the acceleration components in the EmpiricalAcceleration class

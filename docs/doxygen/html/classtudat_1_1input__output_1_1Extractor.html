Detailed Description

template<class T>
class tudat::input_output::Extractor< T >

Extractor base class.

This abstract class belongs to the parser-extractor architecture implemented in Tudat. This base class can be used to derive specific extractors that take data in the form of a ParsedDataLineMapPtr object and return extracted values.

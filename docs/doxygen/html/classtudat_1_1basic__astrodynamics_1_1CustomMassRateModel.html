Detailed Description

Derived class for determining the rate of change of a body's mass, user-defined by a function pointer.

Derived class for determining the rate of change of a body's mass, user-defined by a function pointer. This class can be used for any kind of mass rate model for which the user can define the dependency as a function of time.

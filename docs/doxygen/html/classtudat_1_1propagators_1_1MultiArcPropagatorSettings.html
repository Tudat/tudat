template<typename StateScalarType = double>
class tudat::propagators::MultiArcPropagatorSettings< StateScalarType >

Class for defining setting of a propagator for multi-arc dynamics.

Base class for defining setting of a propagator for multi-arc dynamics. This class contains single-arc propagator settings for each arc.

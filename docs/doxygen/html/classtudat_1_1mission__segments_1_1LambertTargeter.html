virtual void tudat::mission_segments::LambertTargeter::execute ( )
protectedpure virtual

Execute Lambert targeting algorithm.

Executes the Lambert targeting algorithm. Since the parameters of the Lambert routine are set directly in the constructor, the same LambertTargeter object cannot be reused for a different problem. This makes the class more robust, as all parameters are consistent with a single problem at all times. Since each object corresponds to a unique problem, there is no need to call this function more than once. Therefore, this function is protected and run at construction.

Implemented in tudat::mission_segments::ZeroRevolutionLambertTargeterIzzo, tudat::mission_segments::MultiRevolutionLambertTargeterIzzo, tudat::mission_segments::LambertTargeterIzzo, and tudat::mission_segments::LambertTargeterGooding.

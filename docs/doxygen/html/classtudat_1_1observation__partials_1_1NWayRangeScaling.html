Detailed Description

Derived class for scaling three-dimensional position partial to n-way range observable partial.

Derived class for scaling three-dimensional position partial to n-way range observable partial. Implementation is taken from Moyer(2000) for constituent one-way ranges, whicha re scaled (at O(v/c)) to n-way range implementation

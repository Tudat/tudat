Detailed Description

This class provides the mass rate from a (set of) thrust model(s), used in numerically propagating a body's total mass.

This class provides the mass rate from a (set of) thrust model(s, used in numerically propagating a body's total mass. The mass rate is directly obtained from the thrust accelerations. This class can be used to sum the mass rates due to any number of thrust acceleration models.

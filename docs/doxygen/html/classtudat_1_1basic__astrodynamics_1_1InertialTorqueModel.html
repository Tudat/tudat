Detailed Description

Function to compute the inertial torque term.

Function to compute the inertial torque term, which is included as the angular velocity rate is computed in a body-fixed frame.

Update member variables used by the torque model.

Updates member variables used by the torque model. Function pointers to retrieve the current values of quantities from which the torque is to be calculated are set by constructor. This function calls them to update the associated variables to their current state.

Detailed Description

Derived class for scaling three-dimensional position partial to one-way doppler observable partial.

Derived class for scaling three-dimensional position partial to one-way doppler observable partial. Implementation is taken from Moyer(2000) and is separately implemented for fixed receiver and transmitter.

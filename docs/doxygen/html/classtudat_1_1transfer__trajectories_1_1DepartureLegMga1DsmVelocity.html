Detailed Description

Departure Leg class of an MGA-1DSM velocity formulation trajectory model.

A class that calculates the required impulses for a departure leg for an MGA-1DSM velocity formulation trajectory model. This model specifies the trajectory by defining the initial hyperbolic excess velocity (in 3D) and the moment of application of the DSM. The leg is split in two sublegs. The first subleg is analyzed using a kepler propagator, whereas the second leg is analyzed using a lambert targeter. Typically the user will be primarily interested in the total delta V and the arrival velocity of the spacecraft before the arrival planet.

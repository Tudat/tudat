Detailed Description

template<typename IndependentVariableType, typename DependentVariableType>
class tudat::interpolators::Interpolator< IndependentVariableType, DependentVariableType >

Base class for interpolator.

Base class for the interpolators included in Tudat, the dependent and independent variable types are specified as user parameters, as are the number of dimensions. The number of dimensions is chosen as a template parameter, so that a boost multi_array of this dimension can be used as a member variable of derived classes.

Template Parameters
IndependentVariableTypeType of independent variable(s).
IndependentVariableTypeType of dependent variable.
numberOfDimensionsNumber of independent directions for independent variables.

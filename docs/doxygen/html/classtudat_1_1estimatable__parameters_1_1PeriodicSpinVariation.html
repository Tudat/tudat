Detailed Description

Interface class for estimation of a body's periodic spin variation (for a full planetary rotational model).

Interface class for estimation of a body's periodic spin variation (for a full planetary rotational model). Interfaces the estimation with the periodic spin variation (rotational rate corrections) of a PlanetaryRotationModel object

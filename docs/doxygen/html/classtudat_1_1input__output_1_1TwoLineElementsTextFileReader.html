std::multimap< int, std::string > tudat::input_output::TwoLineElementsTextFileReader::checkTwoLineElementsFileIntegrity ( )

Checks the integrity of the TLE input file.

Checks the integrity of the TLE input file. For instance: line number 1 = 1 and 2 = 2, object identification number of line 1 = line 2, U for unclassified data, modulo-10 checksum check, etc. Deletes objects whose TLE data are corrupted.

Returns
0 for success, integer with number of corrupted objects for failure (with error output).

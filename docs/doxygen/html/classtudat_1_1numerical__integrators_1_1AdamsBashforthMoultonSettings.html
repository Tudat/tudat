template<typename IndependentVariableType = double>
class tudat::numerical_integrators::AdamsBashforthMoultonSettings< IndependentVariableType >

Class to define settings of variable step ABAM numerical integrator.

Class to define settings of variable step ABAM numerical integrator, for instance for use in numerical integration of equations of motion/ variational equations.

initialTimeStart time (independent variable) of numerical integration.
initialTimeStepInitial time (independent variable) step used in numerical integration. Adapted during integration.
minimumStepSizeMinimum step size for integration. Integration stops (exception thrown) if time step comes below this value.
maximumStepSizeMaximum step size for integration.
relativeErrorToleranceRelative error tolerance for step size control.
absoluteErrorToleranceAbsolute error tolerance for step size control.
minimumOrderMinimum order of integrator (default 6).
maximumOrderMaximum order of integrator (default 11).
saveFrequencyFrequency at which to save the numerical integrated states (in units of i.e. per n integration time steps, with n = saveFrequency).
assessTerminationOnMinorStepsWhether the propagation termination conditions should be evaluated during the intermediate sub-steps of the integrator (true) or only at the end of each integration step (false).
bandwidthMaximum error factor for doubling the stepsize (default: 200).

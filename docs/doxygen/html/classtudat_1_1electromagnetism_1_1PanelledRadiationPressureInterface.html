Detailed Description

Class in which the properties of a panelled radiation pressure acceleration model are stored.

Class in which the properties of a panelled radiation pressure acceleration model are stored and in which the current radiation pressure is calculated based on the source power and geometry, as well as from the boxes-and-wings model used to describe the vehicle undergoing the acceleration.

Detailed Description

Base class for light-time correction settings.

Base class for light-time correction settings. This class is not used for calculations of corrections, but is used for the purpose of defining the light time correction properties. The createLightTimeCorrections function produces the classes that calculate the actual corrections, based on settings in instances of derived LightTimeCorrectionSettings classes.

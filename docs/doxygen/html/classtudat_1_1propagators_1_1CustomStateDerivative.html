Detailed Description

template<typename StateScalarType = double, typename TimeType = double>
class tudat::propagators::CustomStateDerivative< StateScalarType, TimeType >

Model to compute the derivative of a custom state (i.e. a state for which the physical significance is not 'known' to the rest of the code

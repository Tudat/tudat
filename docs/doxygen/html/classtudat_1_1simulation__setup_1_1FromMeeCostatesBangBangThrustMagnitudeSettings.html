Detailed Description

Class for defining settings for bang-bang MEE-costate based thrust magnitude.

Class for defining settings for bang-bang MEE-costate based thrust magnitude. Model details can be found in Kluever (2010) and Boudestijn (2014). The MEE-costates are provided for the five slow elements, as a function of time. Constructors for constant costates, and costates from an interpolator, are also provided.

Detailed Description

template<int ObservationSize = 1>
class tudat::observation_models::ConstantObservationBias< ObservationSize >

Class for a constant absolute observation bias of a given size.

Class for a constant absolute observation bias of a given size. For unbiases observation h and bias A, the biased observation is computed as h + A

Detailed Description

Swingby Leg class of an MGA trajectory model.

A class that calculates the required impulses for a swingby leg for an MGA trajectory model. It extends the swingby leg base class and uses a lambert targeter for finding the required departure and arrival velocities of the spacecraft.

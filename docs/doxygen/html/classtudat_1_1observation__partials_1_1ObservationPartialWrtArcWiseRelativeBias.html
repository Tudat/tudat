Detailed Description

template<int ObservationSize>
class tudat::observation_partials::ObservationPartialWrtArcWiseRelativeBias< ObservationSize >

Class for computing the derivative of any observable w.r.t. an arc-wise constant relative observation bias.

Class for computing the derivative of any observable w.r.t. a n arc-wiseconstant relative observation bias. Note that this partial is distinct from most other ObservationPartial partial derived classes, as its implementation is based on the parameter (arc-wise constant observation bias), not the type of observable: the implementation is identical for each observable.

Detailed Description

template<typename IndependentVariableType = double, typename DependentVariableType = double>
class tudat::filters::FilterBase< IndependentVariableType, DependentVariableType >

Filter class.

Base class for the set up and use of filters.

Template Parameters
IndependentVariableTypeType of independent variable. Default is double.
DependentVariableTypeType of dependent variable. Default is double.

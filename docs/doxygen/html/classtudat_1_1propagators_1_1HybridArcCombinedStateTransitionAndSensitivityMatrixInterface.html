Detailed Description

Interface object of interpolation of numerically propagated state transition and sensitivity matrices for a hybrid of single-and multi-arc estimation (single order is put first in concatenation)

Interface object of interpolation of numerically propagated state transition and sensitivity matrices for a hybrid of single-and multi-arc estimation (single order is put first in concatenation). The single- anf multi-arc given as input must be consistent: the single arc bodies/states must also be included in the multi-arc model, in order to properly generate the coupling terms between single and multi-arc states (see HybridArcVariationalEquationsSolver).

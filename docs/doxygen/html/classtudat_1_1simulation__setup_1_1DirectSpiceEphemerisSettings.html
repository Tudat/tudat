Parameters
frameOriginName of body relative to which the ephemeris is to be calculated (optional "SSB" by default).
correctForStellarAberrationBoolean whether to correct for stellar Aberration in retrieved values of (observed state) (optional "ECLIPJ2000" by defalut).
correctForLightTimeAberrationBoolean whether to correct for light time in retrieved values of (observed state) (optional false by default).
convergeLighTimeAberrationBoolean whether to use single iteration or max. 3 iterations for calculating light time (optional false by default).
frameOrientationOrientatioan of the reference frame in which the epehemeris is to be calculated (optional false by default).
ephemerisTypeType of ephemeris that is to be created, always set to direct_spice_ephemeris when using this class directly. The derived class InterpolatedSpiceEphemerisSettings sets this parameter to a different value. Not to be changed by used (optional direct_spice_ephemeris by default).

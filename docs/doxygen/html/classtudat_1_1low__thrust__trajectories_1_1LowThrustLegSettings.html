Detailed Description

Class defining settings for the low-thrust leg.

Class defining settings for the low-thrust leg. This class is a functional (base) class for settings of low-thrust leg that require no information in addition to their type. Classes defining settings for low-thrust leg requiring additional information must be derived from this class.

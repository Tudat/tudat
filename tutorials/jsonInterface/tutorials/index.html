In these pages of the wiki, you will be taught how to use the JSON Interface by analyzing a few examples. In the first three examples:

  • Unperturbed Earth-orbiting Satellite
  • jsonInterface_tutorials_perturbedOrbitPropagation
  • Perturbed Earth-orbiting Satellite Lifetime Maximisation

it will be shown how to use the json_interface application, so no C++ code will have to be written. In the two latter examples:

  • Apollo Capsule Entry
  • Perturbed Earth-orbiting Satellite Lifetime Maximisation Using a Custom C++ Application

custom C++ applications, which use the JSON Interface, will be written. Some additional example applications can be found in your tudatBundle, in the folder:

tudatBundle/tudat/Tudat/JsonInterface/UnitTests
